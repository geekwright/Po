Class PoHeader

A special PoEntry that represents the header of a GNU gettext style PO or POT file. The header is the first entry of a PO file. It has an empty string as the "msgid" value, and a set of structured strings compose the "msgstr" value. PoHeader exposes these structured strings so that the individual values can be fetched or set by name.
