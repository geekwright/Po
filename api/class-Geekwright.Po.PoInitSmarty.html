PoInitSmarty

PoInitPHP provides 'msginit' like logic which can take a Smarty template file, recognize gettext like function tokens, and capture the translatable strings in a PoFile object. The Smarty functions are expected to be in the format: funcname(msgid="message" msgid_plural="plural message" msgctxt="context") The specifics of the function ('funcname') and the argument names ('msgid', 'msgid_plural', and 'msgctxt') can be specified. A properly initialized Smarty v3 object is required.
